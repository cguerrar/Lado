# Biblioteca de Música para Reels Creator

Esta carpeta contiene las pistas de música para el creador de Reels.

## Estructura

```
biblioteca/
├── covers/           # Imágenes de portada por género
│   ├── pop.jpg
│   ├── hiphop.jpg
│   ├── electronica.jpg
│   ├── latino.jpg
│   ├── reggaeton.jpg
│   ├── lofi.jpg
│   ├── cinematico.jpg
│   ├── rock.jpg
│   ├── indie.jpg
│   ├── ambiental.jpg
│   └── acustico.jpg
├── summer-vibes.mp3
├── feel-good.mp3
├── bright-day.mp3
└── ... (ver MusicaController.cs para lista completa)
```

## Archivos Requeridos

Los archivos de audio deben ser:
- Formato: MP3
- Duración: 60-200 segundos
- Libres de derechos (royalty-free)

## Pistas Definidas

Ver `Controllers/MusicaController.cs` para la lista completa de 26 pistas predefinidas.
